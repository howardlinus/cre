## FizzBuzz

This repository contains a simple solution for the classic **FizzBuzz** problem.

### Run

```bash
python3 fizzbuzz.py
```

### Input

A single integer `n` (where `0 < n < 2 * 10^5`).

### Output

For each integer `i` from `1` to `n`, prints on its own line:

- `FizzBuzz` if `i` is divisible by both 3 and 5
- `Fizz` if `i` is divisible by 3 only
- `Buzz` if `i` is divisible by 5 only
- otherwise prints `i`
